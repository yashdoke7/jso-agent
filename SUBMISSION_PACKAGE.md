# Submission Package

## Mandatory Deliverables

1. Google Doc with complete answers for Part A, Part B, and Part C.
2. GitHub repository link containing prototype and documentation.
3. Access enabled for reviewers.

## Optional Bonus Deliverables

1. Live deployment link on Vercel.
2. Short walkthrough video (2 to 4 minutes).
3. Screenshots of key UI states.

## Recommended Package Structure

- Problem statement and impact
- Agent design and architecture
- Dashboard integration across User, HR Consultant, Super Admin, and Licensing
- Governance and ethical compliance coverage
- Metrics and expected outcomes
- Timeline and rollout plan
- Links: GitHub and optional live demo

## Final Pre-Submission Checklist

- All assignment sections are answered.
- Google Doc has clear headings and concise language.
- GitHub README matches the assignment narrative.
- Public repository does not include confidential assignment answer text.
- Environment secrets are not committed.
- At least one screenshot is included in the doc.
- Optional live link is tested and reachable.
